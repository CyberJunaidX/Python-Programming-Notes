# 📦 Variables in Python

## 📌 What is a Variable?
A variable is used to store data in Python.

```python
name = "John"
age = 25
````

---

## ⚠️ Rules for Variables

* Must start with a letter or `_`
* Cannot start with a number
* Variables are case-sensitive
* Cannot use Python keywords (`if`, `class`, `def`)

---

## ❌ Invalid Example

```python
# This will cause an error
5age = 25
```

---

## 🎯 Naming Conventions

### ✔ Good Practice (snake_case)

```python
user_age = 30
```

### ❌ Bad Practice

```python
ua = 30
```

### ❌ Avoid single-letter variables

```python
x = 10
```

---

## 💬 Comments in Python

Comments are used to explain code.

```python
# This is a comment
```

---

# 🖨️ print() Function

## 📌 What it does

Used to display output in Python.

```python
print("Hello World")
```

---

## 📌 Multiple Values

You can print multiple values by separating them with commas:

```python
print("My colors are", "blue", "green", "red")
```
